## Quick Lookup

For rapid answers — 3-5 bullet points with key stats.

```
**[Audience] in [Market] — Key Insights**

- [Headline behavior/attitude] ([X]%, index: [N])
- [Second notable finding] ([X]%, index: [N])
- [Third finding — contrast or surprise] ([X]%, index: [N])
- [Demographic or contextual detail]

Source: GWI Core [quarter/year], [market], internet users aged 16-64
```

**Example:**

**Gen Z in the UK — Social Media Habits**

- 78% use Instagram daily (index: 142 vs. all internet users)
- TikTok is the fastest-growing platform with 65% monthly usage (index: 189)
- YouTube remains the most-used platform at 91% (index: 112)
- Gen Z are 2.3x more likely than average to follow influencer recommendations

Source: GWI Core Q3-Q4 2024, UK, internet users aged 16-64

```

|-----------|--------|
| Age skew | [Primary age group] |
| Gender split | [M/F/Other %] |
| Income level | [High/Mid/Low, index] |
| Education | [Level, index] |
| Employment | [Status, index] |
| Urban/Rural | [Skew] |

### Digital Life
- **Primary device**: [Device] ([X]%, index: [N])
- **Daily screen time**: [Hours]
- **Top platforms**: [Platform 1] ([X]%), [Platform 2] ([X]%), [Platform 3] ([X]%)
- **Key digital behavior**: [Behavior] (index: [N])

### Media Consumption
- **News source**: [Source] (index: [N])
- **Streaming**: [Service(s)] (index: [N])
- **Social content**: [Type/format preference]
- **Audio**: [Podcasts/music/radio] (index: [N])

### Purchase Behavior
- **Top purchase drivers**: [Driver 1], [Driver 2], [Driver 3]
- **Discovery channels**: [How they find brands]
- **Brand loyalty signals**: [High/Low, key factors]
- **Category spending**: [Notable categories]

### Attitudes & Values
- **Key motivations**: [Value 1], [Value 2]
- **Concerns**: [Concern 1], [Concern 2]
- **Lifestyle markers**: [Distinguishing interests/activities]

### So What?
[2-3 sentences on implications: what this audience portrait means for strategy, messaging, or targeting]

```

## Source Attribution

### Standard Attribution Line

Always include at the bottom of any output:

```
Source: GWI Core [Quarter(s)/Year], [Market(s)], internet users aged 16-64
```

### Full Attribution (for formal reports)

```
Source: GWI Core Survey, [Q1-Q4 Year or specific quarters]. Data represents
internet users aged 16-64 in [market(s)]. Sample size: [N]. GWI surveys are
conducted quarterly across 50+ markets, weighted by age, gender, and education.
All data is self-reported.
```

### When Sample Size Is Known

Always include N when available:
```
Source: GWI Core Q3-Q4 2024, UK, internet users aged 16-64 (N=4,230)
```

### When Combining with Other Sources

```
Sources:
- GWI Core Q3-Q4 2024, UK (audience behaviors and attitudes)
- [Other source] (additional context/validation)

Note: Methodologies differ across sources. GWI data represents the online
population aged 16-64 based on self-reported surveys.
```
